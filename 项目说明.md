# EtsyGo 小铺 — 前端 + 管理后台（演示版）

给老板看「效果」的原型。**支付与 USDT 合约全部是占位**，确定方向后再按 BNB testnet 接入。

## 运行

```bash
cd web
npm run dev      # 启动后访问 http://localhost:3000
```

- 用户端（手机 H5）：`http://localhost:3000/` — 建议用浏览器手机模式或窄窗口查看。
- 管理后台（桌面）：`http://localhost:3000/admin`
- 两端互相切换：用户端「我的 → 切换到管理后台」；后台侧栏底部「查看用户端」。

## 技术栈

Next.js 16（App Router）· React 19 · TypeScript · Tailwind v4。
字体走 Google Fonts：思源宋体（标题）/ Fraunces（数字）/ 马善政（手写店招）/ 思源黑体（正文）。

## 设计方向

「EtsyGo 工坊暖色」：暖羊皮纸 + 赤陶 + 手作质感，把「拥有一家手作小店」的叙事做进每个元素。
签名元素＝**纸吊牌店招**（虚线缝边 + 打孔 + 蜡封状态印章）。

## 已还原的业务规则（来自 raw/基本事实.md）

- 固定 10 档 300~3000U；3000U=独立店（必填 EtsyGo 店铺名）、其余=拼店
- 可提现额度 = 累计投资 ×120% − 已用提现总额
- 手续费：注册满 30 天免、未满 5%；提现窗口每月 1/15 日 10:00–22:00（UTC+7）
- 佣金 10 档 5%~50%，手机端只展示 ≤40%，50% 档后台特殊权限开启
- 提前退出扣本金 30%；冻结状态；EtsyGo 改名留操作日志
